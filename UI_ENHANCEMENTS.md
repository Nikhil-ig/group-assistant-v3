## ğŸ¯ Action Button Logic

### Intelligent Context-Aware Buttons

The bot analyzes the last action performed and suggests the **most logical follow-up actions**:

1. **After BAN:**
   - Undo action (Unban)
   - Escalate (nothing, already max)
   - Related action (Warn as record)
   - Info (View details)

2. **After MUTE:**
   - Undo action (Unmute)
   - Escalate (Ban)
   - Related (Warn)
   - Info (Stats)

3. **After KICK:**
   - Escalate (Ban)
   - Alternative (Mute)
   - Record action (Log)
   - History (Stats)

4. **After PROMOTE:**
   - Undo action (Demote)
   - Customize (Set role)
   - Manage (Permissions)
   - Info (Admin info)

5. **After WARN:**
   - Escalate options (Ban, Mute, Kick)
   - Tracking (Count, Save)
