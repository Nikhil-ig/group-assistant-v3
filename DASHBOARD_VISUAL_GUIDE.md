### Responsive Breakpoints

#### Desktop (1920px)
- 4 columns for stat cards
- 2 columns for group grid
- Full table width

#### Tablet (1024px)
- 2 columns for stat cards
- 1-2 columns for group grid
- Scrollable table

#### Mobile (768px)
- 1 column for stat cards
- 1 column for group grid
- Horizontal scroll tables
