# V3 Moderation Frontend (React + Vite + Tailwind)

Quickstart:

1. cd into the frontend folder:

```bash
cd v3/frontend
```

2. Install dependencies:

```bash
npm install
```

3. Start dev server:

```bash
npm run dev
```

Notes:
- This project uses Vite + React + Tailwind CSS. Tailwind is configured via `tailwind.config.cjs` and PostCSS.
- If you see missing packages, run `npm install` in the `v3/frontend` folder.
